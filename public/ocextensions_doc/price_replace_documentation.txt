Installation
~~~~~~~~~~~~

This installation runs an SQL query automatically - this adds a new table - cart_p - to your database, no other table is affected. However you may want to run a database backup before installation. 

For OCMOD installations please see documentation at

http://ocextensions.co.uk/ocmod/ocmod

Description
~~~~~~~~~~~

Displays the price of a product with the option prices added on the product page.
Price changes with each option selected and quantity.
Price replace extensions include the ability to choose prefix (+ , - ) or no prefix (=) in admin
